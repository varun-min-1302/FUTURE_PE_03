# 📝 Prompt 02 — Supporting Blog Generator (Content Cluster)

> **Type:** Reusable SEO Prompt
> **Purpose:** Generate focused supporting blogs that feed traffic into the pillar page
> **Works With:** Claude, ChatGPT, Gemini

---

## What Is a Supporting Blog?

A supporting blog is a **shorter, focused article** (700–1200 words) that:
- Targets a specific sub-keyword from the main topic
- Answers one narrow question very well
- Links back to the pillar blog
- Captures long-tail search traffic that the pillar can't target alone

---

## ⚙️ How to Use This Prompt

1. Choose which supporting blog topic you're writing (see examples below)
2. Fill in all `[VARIABLES]`
3. Paste into Claude or ChatGPT
4. Generate → review → add internal links to pillar and other supporting blogs

---

## 🔧 Variables to Replace

| Variable | Example Value |
|---|---|
| `[BUSINESS NAME]` | PixelPeak Digital |
| `[BUSINESS TYPE]` | Digital Marketing Agency |
| `[CITY]` | Hyderabad |
| `[SUPPORTING TOPIC]` | SEO Services in Hyderabad |
| `[TARGET KEYWORD]` | SEO services in Hyderabad |
| `[SEARCH INTENT]` | Commercial Investigation |
| `[PILLAR BLOG TITLE]` | Best Digital Marketing Agency in Hyderabad |
| `[WORD COUNT]` | 900 |
| `[TONE]` | Helpful, clear, slightly conversational |

---

## 🤖 The Prompt

```
You are an SEO content specialist writing for [BUSINESS NAME], a [BUSINESS TYPE] in [CITY], India.

Write a focused supporting blog post for the following:

Topic: [SUPPORTING TOPIC]
Target Keyword: [TARGET KEYWORD]
Search Intent: [SEARCH INTENT]
Word Count: [WORD COUNT] words
Tone: [TONE]
Pillar Blog to link back to: "[PILLAR BLOG TITLE]"

CONTENT STRUCTURE:
1. Meta description (under 155 characters) — include target keyword
2. SEO Title Tag (under 60 characters) — include target keyword
3. H1: Use the target keyword naturally
4. Introduction (100 words): State the problem this blog solves
5. 3–5 H2 sections covering the topic in depth
6. Use H3s inside sections where the topic has sub-points
7. Include [CITY] naturally in at least 4 places — mention specific areas (Hitech City, Gachibowli, Kondapur, Banjara Hills, HITEC City) where it fits
8. One internal link to the pillar blog — use anchor text: "[PILLAR BLOG TITLE]"
9. One internal link suggestion to another related supporting blog (describe what it would be)
10. Closing paragraph with a CTA that drives readers to contact [BUSINESS NAME]
11. FAQ section: 3 frequently asked questions with answers

RULES:
- Never keyword stuff — each keyword use must feel natural
- Write for humans first, search engines second
- Be specific. Generic advice = low rankings
- Do not pad content — every sentence must add value
- Use [CITY]-specific references to improve local relevance

Write the full blog now.
```
