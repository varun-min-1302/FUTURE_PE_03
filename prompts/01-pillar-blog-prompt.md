# 📝 Prompt 01 — Pillar Blog Generator

> **Type:** Reusable SEO Prompt
> **Purpose:** Generate a long-form, SEO-optimized pillar blog for any local business
> **Works With:** Claude, ChatGPT, Gemini

---

## ⚙️ How to Use This Prompt

1. Copy the prompt block below
2. Replace all `[VARIABLES]` with your client's details
3. Paste into Claude or ChatGPT
4. Review output → edit brand voice if needed
5. Add internal links manually based on your content cluster

---

## 🔧 Variables to Replace

| Variable | Example Value |
|---|---|
| `[BUSINESS TYPE]` | Digital Marketing Agency |
| `[BUSINESS NAME]` | PixelPeak Digital |
| `[CITY]` | Hyderabad |
| `[STATE]` | Telangana |
| `[PRIMARY KEYWORD]` | digital marketing agency in Hyderabad |
| `[TARGET AUDIENCE]` | SMB owners and startup founders in Hyderabad |
| `[SERVICES LIST]` | SEO, Social Media Marketing, Google Ads, Content Marketing, Web Design |
| `[WORD COUNT]` | 2000 |
| `[TONE]` | Professional but approachable, helpful, no jargon |

---

## 🤖 The Prompt

```
You are an expert SEO content writer with 10+ years of experience writing for digital marketing agencies and local businesses in India.

Your task is to write a long-form SEO pillar blog post for the following business:

Business Name: [BUSINESS NAME]
Business Type: [BUSINESS TYPE]
Location: [CITY], [STATE]
Primary Keyword: [PRIMARY KEYWORD]
Target Audience: [TARGET AUDIENCE]
Services: [SERVICES LIST]
Desired Word Count: [WORD COUNT] words
Tone: [TONE]

CONTENT REQUIREMENTS:
1. Start with an H1 that includes the primary keyword naturally
2. Write a compelling introduction (150 words) that hooks the reader and states clearly what they'll learn
3. Use H2 and H3 subheadings — include keyword variations naturally in at least 3 headings
4. Include the following sections:
   - What is [BUSINESS TYPE] and why does your business need it
   - Key services offered (with one H3 per service)
   - Why location matters — mention [CITY] areas like key business districts
   - How to evaluate and choose the right agency
   - Real results: what good [BUSINESS TYPE] looks like (use realistic but fictional data)
   - Pricing overview (ballpark ranges, not exact quotes)
   - FAQs section (5 questions and answers)
   - Strong CTA at the end
5. Mention [CITY] and nearby business hubs (e.g., Hitech City, Gachibowli) naturally — not forced
6. Each section must be genuinely helpful — written for the reader first, Google second
7. Add a meta description (155 characters max) at the top of the output
8. Add a suggested SEO title tag (60 characters max) at the top of the output
9. Suggest 3 internal link anchor texts at the bottom (for supporting blogs)
10. Do NOT use bullet-point-heavy lazy writing — use paragraphs, lists only where naturally useful

Write the full blog now. Do not summarize or truncate.
```

---

## 🎛️ Variation: Increase Localization

Add this to the prompt for stronger local SEO targeting:

```
Additionally, mention at least 3 specific neighborhoods or tech hubs in [CITY] where businesses commonly need these services. Reference local industries (IT, pharma, real estate, restaurants) that are common in [CITY].
```

---

## 🎛️ Variation: Add E-E-A-T Signals

Add this block to make the content feel more authoritative (Google's E-E-A-T = Experience, Expertise, Authoritativeness, Trustworthiness):

```
Write the content as if authored by a senior digital marketer with 8+ years of experience in the Indian market. Include specific advice, not generic tips. Reference real frameworks (e.g., AIDA, funnel stages, Core Web Vitals). Cite realistic industry stats naturally within the text.
```

## ✅ Expected Output Checklist

After generating, verify your output has:

- [ ] Meta description (under 155 chars)
- [ ] SEO title tag (under 60 chars)
- [ ] H1 with primary keyword
- [ ] At least 5 H2 sections
- [ ] City mentioned 8–12 times naturally
- [ ] FAQs section (5+ questions)
- [ ] Strong closing CTA
- [ ] Internal link suggestions
- [ ] 1800–2200 word count
